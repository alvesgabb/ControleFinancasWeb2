# ControleFinancasWeb2

**Controle Financeiro pessoal**

Esse projeto é uma aplicação WEB de Controle Financeiro Pessoal que permite o usuário lançar receitas (entradas) e despesas (saídas), e visualizar o saldo atualizado em tempo real.O sistema utiliza uma arquitetura Cliente-Servidor (Full-Stack) para garantir a persistência e a integridade dos dados.

**Arquitetura e Tecnologias:**

**Frontend:** HTML5,CSS3,JavaScript
**Backend:** Node.js, Express.js
**Banco de Dados:** Sqlite

*Como executar a aplicação:*

**Pré requisitos:**
Node.js e npm instalados

**1.** Iniciar o servidor
**1.** Iniciar o Servidor: Navege até a pasta do backend em seu terminal.
**2.** Instale as depêndencias.
**3.** Inicie o servidor. : O servidor deve iniciar e mostrar a mensagem "Servidor rodando em http://localhost:300"

**2.Acessar a interface**
**1.** Com o servidor rodando navege até a pasta do frontend.
**2.** Abra o arquivo index.html no seu navegador e use o Live Server do VS Code

**Funcionalidades Principais:**
Lançamento de novas entradas e saídas (POST)
Visualização instantânea do Saldo Total.
Exclusão de transações individuais (DELETE)
